# 安装Ruby

<p>1.1 安装Ruby<br> 下载ruby: <a href="http://rubyinstaller.org/downloads/" target="_blank">http://rubyinstaller.org/downloads/</a></p><p>我这里下载的是: rubyinstaller-2.1.5-x64.exe</p><p>next, next, next(尽量全选, 选择添加到Path环境变量).</p><p>我自己的安装路径:</p><p>D:\Ruby21-x64</p><p>(如果手动添加Path环境变量, 则变量是D:\Ruby21-x64\bin)</p><p>验证, 输入如下命令行(蓝色字体):</p><p>C:\Users\Administrator>ruby -v</p><p>1.2 修改下载镜像</p><p><a href="http://rubygems.org/" target="_blank">http://rubygems.org/</a> 是被和谐的, 所以要修改下载源, 修改淘宝提供的镜像</p><p>依次输入如下3条命令:</p><p>C:\Users\Administrator>gem sources --remove <a href="http://rubygems.org/" target="_blank">http://rubygems.org/</a></p><p>C:\Users\Administrator>gem sources -a <a href="http://ruby.taobao.org/" target="_blank">http://ruby.taobao.org/</a></p><p>C:\Users\Administrator>gem sources -l</p><p>1.3 输入一堆命令</p><p>更新ruby和gems系统:</p><p>C:\Users\Administrator>gem update --system</p><p>安装rspec组件"</p><p>C:\Users\Administrator>gem install rspec</p><p>安装httparty组件</p><p>C:\Users\Administrator>gem install httparty</p><p>安装wdm组件</p><p>C:\Users\Administrator>gem install wdm</p><a name="安装DevKit"></a>

# 安装DevKit

<p>2.1 解压DevKit</p><p>下载DevKit: <a href="http://rubyinstaller.org/downloads/" target="_blank">http://rubyinstaller.org/downloads/</a></p><p>我这里下载的是: DevKit-mingw64-64-4.7.2-20130224-1432-sfx.exe</p><p>解压DevKit-mingw64-64-4.7.2-20130224-1432-sfx.exe, 我这里解压到目录 D:\DevKit-64.</p><p>2.2 修改配置</p><p>打开D:\DevKit-64\config.yml文件, 修改文件内容为如下: (这里的路径就是对应你的Ruby的安装路径了, 而且要注意格式)</p><p>2.3 安装DevKit和cucumber插件</p><p>打开cmd.exe, CD到D:\DevKit-64, 依次输入如下命令:</p><p>D:\DevKit-64>ruby dk.rb init</p><p>D:\DevKit-64>ruby dk.rb review</p><p>D:\DevKit-64>ruby dk.rb install</p><p>D:\DevKit-64>gem install cucumber</p><a name="安装Jekyll"></a>

# 安装Jekyll

<p>输入如下命令:</p><p>D:\DevKit-64>gem install jekyll</p><a name="安装Pygments"></a>

# 安装Pygments

<p>Jekyll 里默认的语法高亮插件是Pygments.</p><p>Jekyll 还有另一个高亮引擎名为 Rouge(原生Ruby的程序 <a href="https://github.com/jneen/rouge" target="_blank">https://github.com/jneen/rouge</a>).</p><p>Pygments安装步骤:</p><p>4.1 安装Python</p><p>我安装的是Python2.7最新版. <a href="http://www.python.org/download/" target="_blank">http://www.python.org/download/</a></p><p>4.2 安装easy_install</p><p><a href="https://pypi.python.org/pypi/setuptools#installation-instructions" target="_blank">https://pypi.python.org/pypi/setuptools#installation-instructions</a> 来查看详细的安装指南。</p><p>对于 Windows 7 的机器，下载 ez_setup.py 并保存.</p><p>下载地址: <a href="https://bitbucket.org/pypa/setuptools/raw/bootstrap/ez_setup.py" target="_blank">https://bitbucket.org/pypa/setuptools/raw/bootstrap/ez_setup.py</a></p><p>保存后运行:(我保存在D盘)</p><p>C:\Users\Administrator>python “D:\ez_setup.py”</p><p>添加目录"D:\Python27\Scripts"到环境变量PATH.</p><p>确保 easy_install 已经正确安装, 输入命令:</p><p>C:\Users\Administrator>easy_install --version</p><p>会输出: setuptools xxx</p><p>4.3 安装Pygments</p><p>输入命令:</p><p>C:\Users\Administrator>easy_install Pygments</p><a name="Jekyll使用"></a>

# Jekyll使用

<p>依次输入如下命令:</p><p>C:\Users\Administrator>D:</p><p>D:>jekyll new TestJekyll</p><p>D:>cd TestJekyll</p><p>D:\TestJekyll>jekyll serve</p><p>这里输出报了一个警告</p><p>D:/Ruby21-x64/lib/ruby/gems/2.1.0/gems/posix-spawn-0.3.9/lib/posix/spawn.rb:164:</p><p>warning: cannot close fd before spawn 'which' 不是内部或外部命令，也不是可运行的程序</p><p>或批处理文件。</p><p>(但可以正常运行.)</p><p>然后在浏览器输入地址: <a href="http://localhost:4000/" target="_blank">http://localhost:4000/</a> 你就可以看到你想看的.</p><a name="小结"></a>
